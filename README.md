## Course Description

**Introduction to AI Programming** is a foundational course designed to equip students with the essential programming skills required for Artificial Intelligence.  
The course focuses on:

- Python programming principles  
- Logical and analytical problem-solving  
- Data handling techniques relevant to AI applications  

---

## Lab Work

This repository includes all **official lab exercises and practice tasks** performed throughout the semester.  
Each lab is maintained in a separate directory to ensure clarity, modularity, and ease of navigation.

### Covered Topics

- Python fundamentals (variables, data types, input/output)
- Conditional logic (`if`, `elif`, `else`)
- Iterative structures (`for`, `while`)
- Lists, tuples, sets, and dictionaries
- List and dictionary comprehensions
- User-defined functions and modular programming
- Lambda functions
- File handling (read and write operations)
- Exception handling using `try` / `except`
- Introductory data processing techniques for AI
